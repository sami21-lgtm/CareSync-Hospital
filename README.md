# 🏥 CareSync - Modern Hospital Management System

A modern, responsive, and feature-rich hospital management web application designed with a professional medical aesthetic (inspired by the Bangladesh Specialized Hospital standard). Built with HTML5, CSS3, and JavaScript, featuring glassmorphism UI components and a secure staff portal.

---

## ✨ Features

* **Dual Authentication Interface:** 
  * **Staff Login Portal:** Secure access for authorized medical personnel.
  * **Self Enroll Portal:** Quick patient self-registration system.
* **Interactive Dashboard:** 
  * Real-time statistics tracking for total patients, active doctors, and available beds.
  * Clean, modern sidebar navigation with user profile integration.
* **Dynamic Patient Management:** 
  * Add, update, and manage patient records dynamically.
  * Department-based doctor selection and automated tracking.
* **Fully Responsive Design:** 
  * Optimized layout seamlessly adapting to mobile phones, tablets, and desktop computers.

---

## 🛠️ Technology Stack

* **Markup:** HTML5
* **Styling:** CSS3 (Flexbox, Grid, Glassmorphism Effects, Custom Variables)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Icons & Fonts:** FontAwesome & Google Fonts ('Poppins')

---

## 🚀 Getting Started

To run this project locally on your machine, follow these simple steps:

1. Clone or download this repository to your local computer.
2. Ensure you have the following files structured in your project directory:
   * `index.html`
   * `style.css`
   * `script.js`
3. Open the `index.html` file using any modern web browser (Google Chrome, Microsoft Edge, etc.) or use **Live Server** in VS Code.
